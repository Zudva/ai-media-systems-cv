## Public Repository Model

This repository is the stable public hub. Individual code demos will be linked
from here when they pass their own publication gates.

Current public-safe model:

- publish compact proof and method descriptions here;
- publish separate clean-room demo repositories for code examples;
- keep private implementation repos, datasets, media, logs, keys, and customer
  details out of public history;
- use conservative provenance wording: tamper-evidence, manifests, hashes,
  verification gates, and C2PA-compatible Content Credentials framing.

The material-protection work is described as a verification and provenance
method. It does not claim to prevent copying, prove copyright ownership, or
make assets legally authentic by itself.
