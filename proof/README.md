# Proof Pagers

Curated public proof pagers:

- `skills-onepager-en.pdf`
- `skills-onepager-ru.pdf`
- `skills-stackmap-en.pdf`
- `skills-stackmap-ru.pdf`
- [`evidence-matrix.md`](evidence-matrix.md)

These PDFs summarize the relationship between applied skills and project
families. They are intentionally compact and public-facing; deeper evidence,
commit maps, and private review notes are kept outside this public repository.

The evidence matrix is the public Markdown index for the same idea: skills,
current artifacts, evidence tier, and the next proof item needed to make a claim
stronger.
